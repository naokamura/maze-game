# Maze Game 🎮

Django と JavaScript(Canvas) を使って作った  
ブラウザで遊べる 2D 迷路ゲームです。

ランダム生成される迷路を進み、敵キャラを避けながらゴールを目指します。

---

## 🎯 特徴

- ランダム迷路生成（毎回違うマップ）
- Canvas を使った 2D 描画
- プレイヤー移動（キーボード操作）
- 敵キャラクターの自動移動
- アニメ風の見た目・演出

---

## 🎮 操作方法

- ↑ ↓ ← → キー：プレイヤー移動
- リセットボタン：迷路を再生成

---

## 🛠 使用技術

- Python
- Django
- JavaScript
- HTML / CSS
- Canvas API

---
